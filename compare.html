Which phase bar feels right?

Three designs for the S·T·A·C·K phase breakdown column. Same data, same table — only the bar changes.

ℹ️ In all three options: Bottleneck moved to the expanded row · ETA shows date only · Trend replaced by a simple ↑ / → indicator next to the score · Column count reduced from 7 to 5.
A
Pure Color — No Text Inside Bars
S · T · A · C · K labels sit once in the column header. Bars are pure color — the score tells you the number.
Pros
  • Cleanest, most minimal look
  • Color tells the story instantly — no reading required
  • Labels never repeat — zero visual clutter
Cons
  • Have to glance at the header to know which block is which phase
  • Less immediately scannable without labels

B
Ghost Letters — Low Contrast, Same Hue
S · T · A · C · K letters are embedded inside each bar at very low opacity — visible if you look, invisible if you don't. No numbers.
Pros
  • Labels are there without dominating — best of both worlds
  • Still mostly color-read but orientation is easier
  • Feels premium — like a watermark
Cons
  • Slightly more "designed" — might feel busy to some
  • Ghost on yellow (score 2) is hardest to see

C
Segmented Pill — Single Compact Bar
Five color segments in one thin pill. S · T · A · C · K labels sit beneath the pill. Maximum density — every relationship's pattern visible at a glance.
Pros
  • Most compact — fits naturally in a narrow column
  • Pattern recognition is instant — red gaps pop out
  • Leaves room for more columns or larger font elsewhere
Cons
  • Thinner bars are harder to hover/interact with
  • Less visually impactful than a full block bar
